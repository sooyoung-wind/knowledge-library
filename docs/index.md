## Why This Library

한 번 읽고 잊히는 자료가 아니라, 시간이 지나도 계속 가치가 남는 문서를 모으는 것이 목표입니다.  
재생에너지처럼 지식도 축적하고 순환할수록 효율이 좋아집니다. 이 홈은 그런 흐름이 바로 보이도록 설계했습니다.

## How To Expand

1. 새 Markdown 문서는 `docs/` 아래 주제별 폴더에 추가합니다.
2. 단일 페이지 HTML 문서는 `docs/html/` 아래에 보관합니다.
3. `mkdocs.yml`의 `nav`에 링크를 추가해 입구를 정리합니다.
4. GitHub에 push 하면 GitHub Pages가 라이브러리를 자동 갱신합니다.
